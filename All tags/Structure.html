      <meta charset="utf-8">

      <header>  Some header info: logo,tag line, etc.(sometimes, navigation)
        <nav></nav>  Contains links to different parts of web site - Navigation
      </header>
 
      <h1></h1>  Main Heading of the Page

        Has a part of info as articles, divs, spans
      </section>
